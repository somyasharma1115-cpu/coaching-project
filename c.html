🐚 Welcome to C Programming

C is the mother of many modern languages. It's powerful, fast, and used in operating systems, embedded devices, and game engines. This course explains everything in simple words with real examples. No stress — just code!

📌 1. Your First C Program

Every C program starts with main(). Let's print "Hello World" to the screen.

#include <stdio.h>

int main() {
    printf("Hello, welcome to C!\n");
    return 0;
}
🔍 Simple explanation:
• #include <stdio.h> → includes the standard input/output library (needed for printf).
• int main() → the starting point of any C program.
• printf(...) → prints text to the screen.
• \n → moves to a new line (like pressing Enter).
• return 0; → tells the system "program finished successfully".
💡 Try it: Change the text inside printf to your own message!

📦 2. Variables – Storing Data

Variables are containers in memory. C has different types for different kinds of data.

int age = 21; // whole number
float price = 19.99; // decimal number (less precise)
double pi = 3.14159; // decimal (more precise)
char grade = 'A'; // single character (uses single quotes)

printf("Age: %d, Price: %.2f, Grade: %c", age, price, grade);
📖 What each type means:
• int → stores integers like 0, 100, -50.
• float / double → stores decimal numbers.
• char → stores one character like 'X', '7', '#'.
🧠 Format specifiers: %d for int, %f for float, %c for char, %s for string.

⌨️ 3. Getting Input from User

Use scanf to read what the user types.

int number;
printf("Enter a number: ");
scanf("%d", &number);
printf("You entered: %d\n", number);
🎤 How it works:
• scanf("%d", &number); → waits for the user to type an integer and stores it in number.
• The & symbol gives the memory address of the variable (don't worry, just remember to use it for now).

⚡ 4. Making Decisions – if / else

Your program can take different actions based on conditions.

int marks = 75;

if (marks >= 60) {
    printf("Passed! Great job.\n");
} else {
    printf("Needs improvement.\n");
}
🧠 Logic: If the condition inside ( ) is true, the first block runs; otherwise, the else block runs.
🧪 Try changing marks to 45 and see the output.

🔄 5. Loops – Repeat Actions

Loops help you run the same code multiple times without writing it again.

// for loop – when you know how many times
for (int i = 1; i <= 5; i++) {
    printf("Count: %d\n", i);
}

// while loop – repeats as long as condition is true
int j = 1;
while (j <= 3) {
    printf("While loop: %d\n", j);
    j++;
}
🔁 For loop explained:
• int i = 1 → start at 1.
• i <= 5 → continue while i is less or equal to 5.
• i++ → increase i by 1 each time.

📊 6. Arrays – Lists of Data

An array stores many values of the same type in a single variable.

int scores[5] = {90, 85, 78, 92, 88};

for (int i = 0; i < 5; i++) {
    printf("Score %d: %d\n", i+1, scores[i]);
}
📌 Key points:
• Indexing starts at 0: scores[0] is the first element (90).
• Use loops to go through arrays easily.

🧩 7. Functions – Reusable Code Blocks

Functions help you organize code and avoid repetition.

// function definition
void greet() {
    printf("Hello from a function!\n");
}

int add(int a, int b) {
    return a + b;
}

int main() {
    greet();
    int sum = add(5, 3);
    printf("Sum is %d\n", sum);
    return 0;
}
📚 What's happening?
• void greet() → a function that does not return a value.
• int add(int a, int b) → takes two integers and returns their sum.
• In main() we call the functions.

📍 8. Pointers – Memory Addresses

A pointer stores the address of a variable. It sounds advanced, but it's just a number that tells where data lives.

int value = 42;
int *ptr = &value; // ptr stores the address of 'value'

printf("Value: %d\n", value); // prints 42
printf("Address: %p\n", ptr); // prints address (hex)
printf("Via pointer: %d\n", *ptr); // prints 42 (dereferencing)
🧭 Simple analogy: Think of a variable as a house. The pointer is the address of that house. *ptr lets you enter the house and see what's inside.
💡 Pointers are very useful for arrays, functions, and dynamic memory. Don't worry if it's tricky at first — practice!

🔤 9. Strings – Working with Text

In C, strings are arrays of characters ending with a null character '\0'.

char name[] = "Alice"; // automatically adds '\0'
printf("Hello, %s\n", name);

// modify string
name[0] = 'B';
printf("New name: %s\n", name); // prints "Blice"
📝 Important: Strings are just arrays of characters. Use %s to print them.

🧮 Full Example – Simple Calculator

This program asks for two numbers and prints their sum.

#include <stdio.h>

int main() {
    double num1, num2;
    printf("Enter first number: ");
    scanf("%lf", &num1);
    printf("Enter second number: ");
    scanf("%lf", &num2);
    printf("Sum = %.2f\n", num1 + num2);
    return 0;
}
🎯 Enhance it: Try adding subtraction, multiplication, division!

🔹 if / else → decisions
🔹 for / while → loops
🔹 int, float, double, char → data types
🔹 array[ ] → list of items
🔹 function() → reusable code
🔹 *pointer → memory address
