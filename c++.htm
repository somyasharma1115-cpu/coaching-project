✨ Welcome to C++ Programming ✨

C++ is a powerful language used to make games, apps, and even operating systems. Don't worry — we'll learn it step by step, with simple examples and clear explanations. No prior experience needed!

📌 1. Your First C++ Program

Let's write a tiny program that prints a message. Every C++ program needs a special function called main(). The computer starts from there.

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, I'm learning C++!";
    return 0;
}
🔍 Explanation:
• #include <iostream> → gives us the power to print text.
• using namespace std; → lets us write cout instead of std::cout.
• cout << "..." → prints text to the screen.
• return 0; → tells the program "everything finished fine".
💡 Try it: Change the text inside the quotes and see what happens!

📦 2. Variables – Storing Information

Variables are like labeled boxes where we store numbers, words, or true/false values.

int age = 16; // whole number
double price = 19.99; // decimal number
char grade = 'A'; // single character
string name = "Sophia"; // text (needs #include <string>)
bool passed = true; // true or false

cout << "Name: " << name << ", Age: " << age;
📖 Simple words:
• int → stores integers like 5, 100, -3.
• double → stores decimals like 3.14.
• string → stores words or sentences.
• bool → stores true/false.

⚡ 3. Making Decisions with if/else

Your program can take different paths depending on a condition.

int score = 85;

if (score >= 60) {
    cout << "You passed! 🎉";
} else {
    cout << "Keep studying!";
}
🧠 How it works: If the condition inside ( ) is true, the first block runs. Otherwise, the else block runs.
⭐ Try changing the score to 45 and see what message appears.

🔄 4. Loops – Doing Things Repeatedly

When you want to print "Hello" 5 times, a loop saves the day.

for (int i = 1; i <= 5; i++) {
    cout << "Hello number " << i << endl;
}
🔁 Explanation:
• int i = 1 → start counting from 1.
• i <= 5 → keep going as long as i is less or equal to 5.
• i++ → increase i by 1 each time.
• The loop will repeat 5 times.
🧪 Challenge: Change i <= 5 to i <= 10 and see the output!

🧩 5. Functions – Group Code into Blocks

Functions let you write code once and use it many times, like a recipe.

// define a function
void sayHello() {
    cout << "Hello there!" << endl;
}

int main() {
    sayHello(); // call the function
    sayHello(); // call it again
    return 0;
}
📚 Simple terms:
• void means the function doesn't return a value.
• sayHello() is the function name.
• In main() we "call" the function by writing its name followed by ().

🏗️ 6. Classes – Blueprints for Objects

A class describes what something has (attributes) and does (methods).

class Car {
public:
    string brand;
    int year;

    void honk() {
        cout << "Beep beep!";
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2022;
    myCar.honk();
    return 0;
}
🚗 What's happening?
• The Car class is a blueprint. It has brand, year, and a honk() action.
• myCar is an actual car object created from that blueprint.
• We use dot . to access the object's properties and actions.

📝 Full Example – Mini Quiz Program

Let's combine everything into a small, useful program.

#include <iostream>
using namespace std;

int main() {
    int answer;
    cout << "What is 5 + 3? ";
    cin >> answer;

    if (answer == 8) {
        cout << "Correct! Well done.\n";
    } else {
        cout << "Oops. The correct answer is 8.\n";
    }
    return 0;
}
🧑‍💻 New thing: cin >> answer reads what the user types and stores it in the variable answer.
🎯 Your turn: Change the question to "10 - 4" and adjust the condition!

📚 Quick Summary (Remember These!)

🔹 cout → prints output: cout << "text";
🔹 cin → gets input: cin >> variable;
🔹 if/else → decision making
🔹 for loop → repetition
🔹 int, double, string, bool → data types
🔹 function → reusable code block
🔹 class → blueprint for objects
🔹 #include → add extra features
