🎨 Complete CSS – Style the Web Beautifully

CSS (Cascading Style Sheets) brings designs to life. From colors and fonts to flexbox, grid, and animations – this course covers every topic with simple explanations and live code. Let's make things beautiful!

🎨 1. CSS Fundamentals

🔹 What is CSS & Syntax

selector {
property: value;
another-property: value;
}
CSS rules: select element, then declare styles. Example: h1 { color: blue; }

🔹 Ways to Add CSS

<link rel="stylesheet" href="style.css"> // external
<style> ... </style> // internal
style="color:red" // inline
External (best for large projects), Internal (single page), Inline (specific element).

🔹 Colors & Units

color: red; background: #00ff00;
border-color: rgb(255,0,0);
width: 50%; margin: 10px; font-size: 1.2rem;
Named, hex, rgb, hsl. Units: px, em, rem, %, vw, vh.

🎯 2. Selectors & Specificity

🔹 Basic Selectors

* { } /* universal */
div { } /* element */
.class { } /* class */
#id { } /* id */
Element selects by tag. Class (.) reusable. ID (#) unique.

🔹 Combinators & Grouping

div p { } /* descendant */
div > p { } /* child */
h1, h2 { } /* grouping */
Space = any descendant. > = direct child. Comma = group selectors.

🔹 Pseudo-classes & Pseudo-elements

a:hover { color:red; }
li:first-child { }
p::first-line { }
input:focus { }
:hover, :active, :nth-child(), ::before, ::after.

🔹 Attribute Selectors

input[type="text"] { }
a[href^="https"] { }
[class~="btn"] { }
Target elements by attributes: ^= starts with, $= ends with, *= contains.

📦 3. Box Model

🔹 Content, Padding, Border, Margin

div {
width: 200px;
padding: 20px;
border: 2px solid black;
margin: 10px;
}
Content inside. Padding inside border. Margin outside. Total width = width + padding + border + margin.

🔹 box-sizing: border-box

* { box-sizing: border-box; }
/* width includes padding & border */
Makes layout predictable. Width includes padding/border.

✍️ 4. Typography & Backgrounds

🔹 Font & Text Properties

font-family: 'Arial', sans-serif;
font-size: 16px; font-weight: bold;
text-align: center; line-height: 1.5;
text-decoration: none; text-transform: uppercase;
Font family, size, weight, line-height, letter-spacing, text align/decoration.

🔹 Backgrounds

background-color: #f0f0f0;
background-image: url('img.jpg');
background-size: cover; background-position: center;
background-repeat: no-repeat;
Color, image, gradient, position, size, attachment.

🏗️ 5. Layout: Display & Position

🔹 Display Property

display: block; // full width
display: inline; // width as content
display: inline-block; // inline but block features
display: none; // hidden (removes space)
Block (div, p), inline (span, a), inline-block (mix).

🔹 Positioning

position: static; // default
position: relative; // relative to normal
position: absolute; // relative to nearest positioned ancestor
position: fixed; // relative to viewport
position: sticky; // scrolls then sticks
Use top, right, bottom, left offsets.

🔹 Z-index & Stacking

z-index: 10; /* only works on positioned elements */
Controls layering (higher number = on top).

🪄 6. Flexbox – One-Dimensional Layout

🔹 Flex Container

.container {
display: flex;
flex-direction: row | column;
justify-content: center | space-between;
align-items: center | stretch;
}
Parent becomes flex container. Controls main axis (justify) and cross axis (align).

🔹 Flex Items

.item {
flex-grow: 1; // take extra space
flex-shrink: 0; // no shrinking
flex-basis: 100px;
align-self: flex-end;
}
flex shorthand: flex: 1 1 auto. order to reorder items.

🧩 7. CSS Grid – Two-Dimensional Layout

🔹 Grid Container

.grid {
display: grid;
grid-template-columns: 1fr 2fr 1fr;
grid-template-rows: auto 200px;
gap: 20px;
}
Define columns and rows. fr = fraction. gap for spacing.

🔹 Grid Item Placement

.item {
grid-column: span 2 / 4;
grid-row: 1 / 3;
}
grid-column: start / end; grid-area with named lines or grid-template-areas.

🔹 Grid Template Areas

grid-template-areas:
"header header"
"sidebar main"
"footer footer";
.header { grid-area: header; }
Visually map layout. Great for responsive.

📱 8. Responsive Design & Media Queries

🔹 Media Queries

@media (max-width: 768px) {
body { font-size: 14px; }
.flex-container { flex-direction: column; }
}
Apply styles based on screen width, height, orientation. Mobile-first: min-width.

🔹 Viewport & Fluid Layouts

<meta name="viewport" content="width=device-width, initial-scale=1">
width: 100%; max-width: 1200px;
Viewport meta for mobile. Use max-width and relative units.

🔹 Mobile First & Breakpoints

/* default small screens */
@media (min-width: 640px) { }
@media (min-width: 1024px) { }
Start with mobile, then add tablet/desktop styles.

✨ 9. Transitions & Animations

🔹 CSS Transitions

.btn {
transition: background 0.3s ease, transform 0.2s;
}
.btn:hover { background: blue; transform: scale(1.05); }
Smooth changes on hover/focus. Properties: all, delay, timing.

🔹 Keyframes Animation

@keyframes slide {
0% { transform: translateX(0); }
100% { transform: translateX(100px); }
}
.box { animation: slide 2s infinite alternate; }
Define keyframes, then apply animation (duration, iteration, direction).

🚀 10. Advanced Topics

🔹 CSS Variables (Custom Properties)

:root { --primary-color: #3498db; }
button { background: var(--primary-color); }
Reusable values, change dynamically with JS or media queries.

🔹 Flexbox vs Grid

Flexbox: 1D (row or column). Grid: 2D (both axes). Use Flex for components, Grid for page layout.

🔹 Transform & 3D

transform: translate(10px,20px) rotate(45deg) scale(1.2);
transform-style: preserve-3d; perspective: 500px;
Translate, rotate, scale, skew. 3D transforms with perspective.

🔹 Pseudo-elements (::before, ::after)

.quote::before { content: "“"; font-size: 2em; }
Insert content without extra HTML. Often used for icons/decorations.

🔹 CSS Filters & Blend Modes

filter: blur(5px) brightness(0.8);
background-blend-mode: multiply;
Filters: blur, grayscale, contrast. Blend modes for backgrounds.

🔹 @import, @font-face, @keyframes

@import url('another.css');
@font-face { font-family: 'MyFont'; src: url('font.woff2'); }
Import external CSS, load custom fonts, define animations.

🔹 CSS Frameworks (concept)

Bootstrap, Tailwind, Foundation – pre-built components and utilities.
