📊 Complete Data Science – From Python to Machine Learning

Data Science combines programming, statistics, and domain knowledge. This course covers Python, NumPy, Pandas, Matplotlib, Seaborn, statistics, data cleaning, and machine learning (scikit-learn). Every topic is explained simply with code examples. Let's unlock the power of data!

🐍 1. Python Core for Data Science

🔹 Variables & Data Types

age = 25 # int
price = 19.99 # float
name = "Alice" # str
is_valid = True # bool
my_list = [1,2,3] # list
my_dict = {"a":1} # dict
Python is dynamically typed. Lists are mutable, tuples immutable, dicts key-value.

🔹 Control Flow & Loops

if score >= 60: print("Pass")
else: print("Fail")

for i in range(5):
print(i)
while condition:
do_something()
if/elif/else, for loops (iterating over sequences), while loops.

🔹 Functions & List Comprehensions

def add(a,b):
return a+b

squares = [x**2 for x in range(10)]
Functions with def. List comprehensions = concise way to create lists.

🔢 2. NumPy – Numerical Computing

🔹 Arrays & Operations

import numpy as np
arr = np.array([1,2,3,4])
zeros = np.zeros((3,3))
ones = np.ones((2,4))
arr + 5, arr * 2, np.sqrt(arr)
NumPy arrays are fast. Vectorized operations (element-wise).

🔹 Indexing, Slicing, Reshaping

arr[0], arr[1:3]
matrix = arr.reshape(2,2)
matrix.T # transpose
np.concatenate([arr1, arr2])
Similar to lists but with multi-dimensional support.

🔹 Universal Functions (ufunc)

np.mean(arr), np.std(arr), np.sum(arr)
np.max(), np.min(), np.dot(a,b)
Statistical and mathematical functions on arrays.

🐼 3. Pandas – Data Manipulation

🔹 Series & DataFrame

import pandas as pd
s = pd.Series([1,2,3])
df = pd.DataFrame({'A':[1,2], 'B':[3,4]})
df.head(), df.info(), df.describe()
Series = 1D labeled array. DataFrame = 2D table like Excel.

🔹 Reading & Writing Data

df = pd.read_csv('data.csv')
df.to_csv('output.csv', index=False)
pd.read_excel('file.xlsx')
pd.read_json('data.json')
Load data from CSV, Excel, JSON, SQL, etc.

🔹 Selecting & Filtering

df['column'] # one column
df[['col1','col2']] # multiple
df[df['age'] > 30] # filtering
df.loc['row_label']
df.iloc[0,1]
loc by label, iloc by index position.

🔹 GroupBy & Aggregation

df.groupby('category')['value'].mean()
df.groupby(['col1','col2']).agg({'col3':'sum', 'col4':'count'})
Split-apply-combine: powerful for summaries.

🔹 Handling Missing Data

df.isnull().sum()
df.dropna()
df.fillna(0)
df.fillna(df.mean())
Detect, remove, or impute missing values (NaN).

🔹 Merge, Join, Concatenation

pd.concat([df1, df2])
pd.merge(df1, df2, on='key')
df1.join(df2, lsuffix='_left')
Combine DataFrames like SQL joins.

📈 4. Data Visualization

🔹 Matplotlib Basics

import matplotlib.pyplot as plt
plt.plot(x, y)
plt.scatter(x, y)
plt.bar(categories, values)
plt.hist(data, bins=20)
plt.title(), plt.xlabel(), plt.show()
Line, scatter, bar, histogram plots. Customize labels/titles.

🔹 Seaborn – Statistical Plots

import seaborn as sns
sns.barplot(x='cat', y='val', data=df)
sns.heatmap(correlation_matrix)
sns.pairplot(df)
sns.distplot(data)
Higher-level interface for attractive stat plots.

🔹 Subplots & Customization

fig, axes = plt.subplots(2,2)
axes[0,0].plot(x1,y1)
plt.figure(figsize=(10,6))
plt.style.use('ggplot')
Multiple plots in one figure, figure size, styles.

📐 5. Statistics Essentials

🔹 Descriptive Stats

mean = np.mean(data)
median = np.median(data)
mode = stats.mode(data)
variance = np.var(data)
std = np.std(data)
percentile = np.percentile(data, 75)
Central tendency, spread, quartiles.

🔹 Probability Distributions

# Normal distribution
import scipy.stats as stats
rvs = stats.norm.rvs(loc=0, scale=1, size=1000)
pdf = stats.norm.pdf(x)
Normal, Binomial, Poisson – used for modeling.

🔹 Hypothesis Testing & Correlation

t_stat, p_val = stats.ttest_ind(group1, group2)
corr_matrix = df.corr()
from scipy.stats import pearsonr
T-test, chi-square, Pearson correlation, p-values.

🤖 6. Machine Learning with scikit-learn

🔹 Train-Test Split & Preprocessing

from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler().fit(X_train)
Split data, scale features, handle categorical (OneHotEncoder).

🔹 Linear Regression

from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)
preds = model.predict(X_test)
from sklearn.metrics import mean_squared_error, r2_score
Predict continuous values. Evaluate using R², MSE.

🔹 Classification (Logistic Regression, Decision Trees)

from sklearn.linear_model import LogisticRegression
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score, confusion_matrix, classification_report
Binary/multi-class classification. Metrics: accuracy, precision, recall, F1.

🔹 Clustering (K-Means)

from sklearn.cluster import KMeans
kmeans = KMeans(n_clusters=3)
kmeans.fit(X)
labels = kmeans.predict(X)
Unsupervised learning – group similar data points.

🔹 Model Evaluation & Cross-Validation

from sklearn.model_selection import cross_val_score, GridSearchCV
scores = cross_val_score(model, X, y, cv=5)
GridSearchCV(model, param_grid, cv=5)
Validate model performance, hyperparameter tuning.

🧹 7. Data Cleaning & Feature Engineering

🔹 Handling Outliers & Duplicates

df.drop_duplicates(inplace=True)
Q1 = df['col'].quantile(0.25)
Q3 = df['col'].quantile(0.75)
IQR = Q3 - Q1
df = df[~((df['col'] < Q1-1.5*IQR) | (df['col'] > Q3+1.5*IQR))]
Remove duplicates, detect outliers using IQR or Z-score.

🔹 Encoding Categorical Variables

df = pd.get_dummies(df, columns=['category'])
from sklearn.preprocessing import LabelEncoder
le = LabelEncoder()
df['encoded'] = le.fit_transform(df['cat'])
Convert categories to numbers (one-hot, label encoding).

🔹 Feature Scaling & Creation

from sklearn.preprocessing import MinMaxScaler
scaler = MinMaxScaler()
df['new_feature'] = df['col1'] / df['col2']
# Polynomial features, interaction terms
Normalize features, create domain-specific features.
