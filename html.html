🌐 Complete HTML – Structure the Web

HTML (HyperText Markup Language) is the foundation of every website. This course covers everything: basic tags, semantic elements, forms, tables, multimedia, accessibility, and more. No prior knowledge needed – just pure HTML from scratch!

🏗️ 1. HTML Fundamentals

🔹 Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>My Page</title>
</head>
<body>
<h1>Hello World!</h1>
</body>
</html>
Doctype declares HTML5. <html> root. <head> metadata, <body> visible content.

🔹 Headings & Paragraphs

<h1>Main title</h1>
<h2>Subheading</h2>
<h6>Smallest heading</h6>
<p>This is a paragraph of text.</p>
<h1>-<h6> define headings (importance decreases). <p> for text blocks.

🔹 Links & Images

<a href="https://example.com">Click me</a>
<img src="photo.jpg" alt="Description" width="200">
<a> hyperlink with href. <img> self-closing, alt for accessibility.

🔹 Lists

<ul>
<li>Unordered item</li>
</ul>
<ol>
<li>Ordered item</li>
</ol>
<dl><dt>Term</dt><dd>Definition</dd></dl>
ul (bullets), ol (numbers), dl (description list).

✍️ 2. Text Formatting & Inline Tags

🔹 Formatting Elements

<strong>Bold important</strong>
<em>Italic emphasis</em>
<mark>Highlight</mark>
<small>Small text</small>
<del>Deleted text</del>
<ins>Inserted text</ins>
<sub>subscript</sub> <sup>superscript</sup>
Semantic vs presentational: <strong> (browsers bold), <em> (italic).

🔹 Inline vs Block

Block elements (div, p, h1) start new line, full width. Inline (span, a, strong) stay in line, width as content. <div> for layout, <span> for inline styling.

🧩 3. Semantic HTML (HTML5)

🔹 Structural Semantic Tags

<header>...</header>
<nav>...</nav>
<main>...</main>
<article>...</article>
<section>...</section>
<aside>...</aside>
<footer>...</footer>
Describe meaning: <header> site header, <nav> navigation, <main> main content, <article> self-contained content, <section> thematic grouping, <aside> sidebar, <footer> footer.

🔹 Why Semantic?

Improves SEO, accessibility (screen readers), code readability. Use instead of generic <div> when possible.

📝 4. Forms – Collecting User Data

🔹 Basic Form Structure

<form action="/submit" method="POST">
<label for="name">Name:</label>
<input type="text" id="name" name="name" required>
<button type="submit">Send</button>
</form>
action: where data goes. method: GET (in URL) or POST (in body). label improves accessibility.

🔹 Input Types

<input type="text">
<input type="email">
<input type="password">
<input type="number">
<input type="date">
<input type="checkbox">
<input type="radio">
<textarea></textarea>
<select><option>...</option></select>
HTML5 provides many specialized input types with validation.

🔹 Form Attributes

placeholder, required, readonly, disabled, maxlength, pattern (regex), autocomplete, novalidate.

🔹 Example: Registration Form

<form>
<input type="text" placeholder="Username" required>
<input type="email" placeholder="Email" required>
<input type="password" placeholder="Password">
<label><input type="checkbox"> Agree to terms</label>
<button>Sign Up</button>
</form>

📊 5. Tables – Displaying Tabular Data

🔹 Table Structure

<table border="1">
<thead><tr><th>Name</th><th>Age</th></tr></thead>
<tbody><tr><td>Alice</td><td>25</td></tr></tbody>
<tfoot><tr><td colspan="2">Total: 1 person</td></tr></tfoot>
</table>
<tr> (row), <th> (header cell), <td> (data cell). colspan, rowspan for merging.

🎬 6. Multimedia – Audio, Video, Images

🔹 Images & Responsive

<img src="image.jpg" alt="description" loading="lazy">
<picture>
<source media="(max-width: 600px)" srcset="small.jpg">
<img src="large.jpg" alt="...">
</picture>
loading="lazy" defers off-screen images. <picture> for art direction.

🔹 Video & Audio

<video controls width="400">
<source src="movie.mp4" type="video/mp4">
Your browser doesn't support video.
</video>

<audio controls>
<source src="song.mp3" type="audio/mpeg">
</audio>
controls attribute shows play/pause. Use multiple sources for different formats.

🔹 Embedding YouTube / Iframe

<iframe width="560" height="315" src="https://www.youtube.com/embed/..." title="YouTube video" allowfullscreen></iframe>
Embed external content (maps, videos, other pages).

🔗 7. Advanced Links & Navigation

🔹 Internal Links / Anchor

<a href="#section2">Go to Section 2</a>
...
<h2 id="section2">Section 2</h2>
id attribute creates target. Same page navigation.

🔹 Email & Telephone Links

<a href="mailto:someone@example.com">Email us</a>
<a href="tel:+1234567890">Call us</a>

⚙️ 8. HTML5 APIs & Advanced Features

🔹 Canvas – Drawing Graphics

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');
ctx.fillStyle = 'red';
ctx.fillRect(10, 10, 50, 50);
</script>
JavaScript drawing API – shapes, images, animations.

🔹 SVG – Scalable Vector Graphics

<svg width="100" height="100">
<circle cx="50" cy="50" r="40" fill="blue" />
</svg>
XML-based vector graphics. Resize without losing quality.

🔹 Local Storage & Session Storage

<script>
localStorage.setItem('key', 'value');
let data = localStorage.getItem('key');
</script>
Store key-value pairs client-side. Persistent across sessions.

🔹 Geolocation, Drag & Drop

Geolocation API: get user location. Drag & Drop: native HTML5 drag events.

♿ 9. SEO & Web Accessibility (a11y)

🔹 SEO Meta Tags

<meta name="description" content="Page description">
<meta name="keywords" content="html, course">
<meta name="author" content="Your Name">
<title>Descriptive Title</title>
Meta description appears in search results. Unique, descriptive titles.

🔹 Accessibility Attributes

<img alt="meaningful description">
<button aria-label="Close">X</button>
<div role="navigation">...</div>
alt for images. ARIA roles for custom widgets. Semantic HTML is best for accessibility.

🔹 Best Practices

Use headings in order. Add lang attribute. Ensure keyboard navigable. Color contrast.

🔣 10. HTML Entities & Special Characters

🔹 Common Entities

&lt; < (less than)
&gt; > (greater than)
&amp; & (ampersand)
&quot; " (double quote)
&copy; © (copyright)
&reg; ® (registered)

🔹 Links & Media
a, img, video, audio, iframe, picture, source
🔹 Lists
ul, ol, li, dl, dt, dd

table, tr, th, td, thead, tbody, tfoot, colspan, rowspan
