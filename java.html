Java is a powerful, object-oriented programming language used for enterprise apps, Android, and backend systems. This course covers everything: syntax, OOP, collections, exceptions, multithreading, file I/O, JDBC, lambdas, and more. Clear examples for all topics. Let's master Java!

📖 1. Java Fundamentals

🔹 Hello World & Setup

public class Main {
public static void main(String[] args) {
System.out.println("Hello Java!");
}
}
JDK + IDE (IntelliJ/Eclipse). javac Main.java, java Main.

🔹 Variables & Data Types

int age = 25;
double price = 19.99;
char grade = 'A';
boolean isValid = true;
String name = "Alice";
Primitive: int, double, char, boolean. Reference: String, arrays, objects.

🔹 Operators & Control Flow

if (score >= 60) {
System.out.println("Pass");
} else {
System.out.println("Fail");
}

for (int i=0; i<5; i++) { ... }
while (condition) { ... }
switch(day) { case 1: ... }

🧩 2. Object-Oriented Programming

🔹 Classes & Objects

class Car {
String brand;
void honk() {
System.out.println("Beep!");
}
}
Car myCar = new Car();
myCar.brand = "Toyota";
myCar.honk();

🔹 Constructors & this

class Student {
String name;
Student(String name) {
this.name = name;
}
}

🔹 Encapsulation (Getters/Setters)

class Person {
private int age;
public int getAge() { return age; }
public void setAge(int age) { this.age = age; }
}
Hide data, control access.

🔹 Inheritance & super

class Animal { void sound() { } }
class Dog extends Animal {
void sound() { System.out.println("Bark"); }
}

🔹 Polymorphism (Overloading & Overriding)

// Method overloading
int add(int a, int b)
double add(double a, double b)

// Overriding (runtime polymorphism)

🔹 Abstract Classes & Interfaces

abstract class Shape { abstract void draw(); }
interface Drawable { void draw(); }
class Circle implements Drawable { ... }
Interfaces for multiple inheritance, abstraction.

📊 3. Arrays & Collections

🔹 Arrays

int[] numbers = {1,2,3};
String[] names = new String[5];
numbers[0] = 10;
for (int n : numbers) { }

🔹 ArrayList, LinkedList

import java.util.*;
List<String> list = new ArrayList<>();
list.add("A");
list.get(0);
list.remove(0);
LinkedList<Integer> linked = new LinkedList<>();
ArrayList: dynamic array. LinkedList: doubly-linked.

🔹 HashSet, TreeSet

Set<String> set = new HashSet<>();
set.add("apple");
set.contains("apple");
TreeSet maintains sorted order.

🔹 HashMap, TreeMap

Map<String, Integer> map = new HashMap<>();
map.put("John", 25);
int age = map.get("John");
TreeMap keeps keys sorted.

🔹 Iterators & for-each

Iterator<String> it = list.iterator();
while(it.hasNext()) { ... }
list.forEach(item -> System.out.println(item));

⚠️ 4. Exception Handling

🔹 try-catch-finally

try {
int x = 10/0;
} catch (ArithmeticException e) {
System.out.println(e.getMessage());
} finally {
System.out.println("Always runs");
}

🔹 throw & throws

void validate(int age) throws Exception {
if (age < 18) throw new Exception("Invalid");
}
Checked vs unchecked exceptions.

🔹 Custom Exceptions

class MyException extends Exception {
MyException(String msg) { super(msg); }
}

⚙️ 5. Multithreading

🔹 Thread class & Runnable

class MyThread extends Thread {
public void run() { ... }
}
MyThread t = new MyThread();
t.start();

Runnable r = () -> System.out.println("run");
new Thread(r).start();

🔹 Synchronization & Locks

synchronized(this) { ... }
// or synchronized method
Lock lock = new ReentrantLock();
lock.lock(); try { } finally { lock.unlock(); }

🔹 Executor Framework

ExecutorService executor = Executors.newFixedThreadPool(5);
executor.submit(() -> doWork());
executor.shutdown();

📁 6. File I/O (java.io & java.nio)

🔹 Reading/Writing Text Files

// Write
try (BufferedWriter bw = new BufferedWriter(new FileWriter("file.txt"))) {
bw.write("Hello");
}
// Read
try (BufferedReader br = new BufferedReader(new FileReader("file.txt"))) {
String line = br.readLine();
}

🔹 Files & Paths (NIO)

Path path = Paths.get("file.txt");
List<String> lines = Files.readAllLines(path);
Files.writeString(path, "content");

🗄️ 7. JDBC – Database Access

🔹 Connecting & Querying

Class.forName("com.mysql.cj.jdbc.Driver");
Connection conn = DriverManager.getConnection(url, user, pass);
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM users");
while(rs.next()) {
String name = rs.getString("name");
}
PreparedStatement pstmt = conn.prepareStatement("INSERT INTO ...");
pstmt.setString(1, "value");
pstmt.executeUpdate();

✨ 8. Lambda Expressions & Streams

🔹 Lambda Syntax

(params) -> expression
List<String> list = Arrays.asList("a","b");
list.forEach(s -> System.out.println(s));

Comparator<String> comp = (s1,s2) -> s1.compareTo(s2);

🔹 Stream API

list.stream()
.filter(s -> s.startsWith("a"))
.map(String::toUpperCase)
.collect(Collectors.toList());

int sum = numbers.stream().mapToInt(i->i).sum();
